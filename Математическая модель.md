### 2.2. –ú–æ–¥–µ–ª—å –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏—è —á–∞—Å—Ç–∏—Ü

## 3. –ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –º–æ–¥–µ–ª—å —Ç–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ –∞–Ω–∞–ª–∏–∑–∞

### 3.1. –í–µ–∫—Ç–æ—Ä –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ —Å–æ–±—ã—Ç–∏—è

- $f_4 = p_{y,\text{total}}$ - —Å—É–º–º–∞—Ä–Ω—ã–π –∏–º–ø—É–ª—å—Å –ø–æ y

### 3.2. –ú–∞—Ç—Ä–∏—Ü–∞ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–π

–ú–∞—Ç—Ä–∏—Ü–∞ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–π –º–µ–∂–¥—É —Å–æ–±—ã—Ç–∏—è–º–∏:

–ì–¥–µ $\mu$ –∏ $\sigma$ - —Å—Ä–µ–¥–Ω–µ–µ –∏ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–µ –ø–æ –∫–∞–∂–¥–æ–º—É –ø—Ä–∏–∑–Ω–∞–∫—É, $\epsilon = 10^{-12}$ - –∑–∞—â–∏—Ç–∞ –æ—Ç –¥–µ–ª–µ–Ω–∏—è –Ω–∞ –Ω–æ–ª—å.

### 3.3. –ü–µ—Ä—Å–∏—Å—Ç–µ–Ω—Ç–Ω–∞—è –≥–æ–º–æ–ª–æ–≥–∏—è

–î–ª—è –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è –†–∏–ø—Å-–∫–æ–º–ø–ª–µ–∫—Å–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏—è –ø–æ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—é $r$:

$$\text{–†–µ–±—Ä–æ } (i,j) \in \text{–∫–æ–º–ø–ª–µ–∫—Å} \iff D_{ij} \leq r$$

–î–∏–∞–≥—Ä–∞–º–º–∞ –ø–µ—Ä—Å–∏—Å—Ç–µ–Ω—Ç–Ω–æ—Å—Ç–∏ –¥–ª—è –≥–æ–º–æ–ª–æ–≥–∏–π —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç–∏ $k$:

$$\text{–î–∏–∞–≥—Ä–∞–º–º–∞}_k = \{(b_i, d_i) | i = 1, 2, ..., n_k\}$$

–ì–¥–µ $b_i$ - –≤—Ä–µ–º—è —Ä–æ–∂–¥–µ–Ω–∏—è, $d_i$ - –≤—Ä–µ–º—è —É–Ω–∏—á—Ç–æ–∂–µ–Ω–∏—è $i$-–≥–æ —Ç–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–∏–∑–Ω–∞–∫–∞.

–ß–∏—Å–ª–∞ –ë–µ—Ç—Ç–∏:

$$\beta_k = \text{–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç —Å–≤—è–∑–Ω–æ—Å—Ç–∏ –≤ –≥–æ–º–æ–ª–æ–≥–∏—è—Ö —Ä–∞–∑–º–µ—Ä–Ω–æ—Å—Ç–∏ } k$$

### 3.4. –ê–Ω–∞–ª–∏–∑ –∫–æ—Ä—Ä–µ–ª—è—Ü–∏–π –∏ PCA

–ö–æ—Ä—Ä–µ–ª—è—Ü–∏–æ–Ω–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞:

$$C_{ij} = \frac{\text{cov}(f_i, f_j)}{\sigma_i \sigma_j}$$

–°–ø–µ–∫—Ç—Ä –∫–æ—Ä—Ä–µ–ª—è—Ü–∏–æ–Ω–Ω–æ–π –º–∞—Ç—Ä–∏—Ü—ã:

$$C = V \Lambda V^T$$

–ì–¥–µ $\Lambda = \text{diag}(\lambda_1, \lambda_2, ..., \lambda_d)$ - –¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π, —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã—Ö –ø–æ —É–±—ã–≤–∞–Ω–∏—é.

–£—Å–ª–æ–≤–Ω—ã–π –Ω–æ–º–µ—Ä:

$$\kappa = \frac{\lambda_{\text{max}}}{\lambda_{\text{min}} + \epsilon}$$

PCA –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ:

$$\mathbf{z} = V^T \mathbf{f}_{\text{norm}}$$

–û–±—ä—è—Å–Ω–µ–Ω–Ω–∞—è –¥–∏—Å–ø–µ—Ä—Å–∏—è –¥–ª—è $k$ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç:

$$\text{–î–æ–ª—è –æ–±—ä—è—Å–Ω–µ–Ω–Ω–æ–π –¥–∏—Å–ø–µ—Ä—Å–∏–∏} = \frac{\sum_{i=1}^k \lambda_i}{\sum_{i=1}^d \lambda_i}$$

## 4. –ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –º–æ–¥–µ–ª—å –∫–∞–ª–∏–±—Ä–æ–≤–∫–∏

### 4.1. –§—É–Ω–∫—Ü–∏—è –æ—à–∏–±–∫–∏

$$L(\theta) = \sqrt{\frac{1}{N}\sum_{i=1}^{N} w_i \left(\frac{y_i(\theta) - y_i^*}{\max(|y_i^*|, \epsilon)}\right)^2}$$

–ì–¥–µ:
